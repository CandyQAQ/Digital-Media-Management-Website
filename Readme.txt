1. 下载所有文件并解压，放在新建的文件夹HW2中，将HW1中的Soft和Noisy文件夹
放在Music/src/assets文件夹中，与其他图片一起
2. 按照下述方式组织文件：
-HW2
    -.idea
    -mysite（新建的目录文件夹）
        -Music
        -mymusic
        -mysite（下载下来的文件夹）
        -node_modules
        -db.sqlite3
        -manage.py
    -venv
    -Readme
    -起服务器说明
    -网站功能展示
3. 用pycharm打开文件
4. 进入mysite目录python manage.py runserver运行
5. 打开另一个local终端窗口
6. 进入Music目录cnpm install，然后cnpm run dev，等待程序运行后用前端url打开
